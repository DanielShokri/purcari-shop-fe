# Purcari Admin Dashboard

Hebrew RTL admin dashboard built with React 19, TypeScript, Chakra UI v3, and Appwrite.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Tech Stack

- React 19 + TypeScript + Vite
- Chakra UI v3
- Redux Toolkit + RTK Query
- Appwrite SDK
- React Router v7
