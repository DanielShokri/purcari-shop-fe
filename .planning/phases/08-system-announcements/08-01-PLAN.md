  <action>
    Create `convex/systemAnnouncements.ts` with the following functions (follow patterns from convex/coupons.ts):

    1. **list** (query): 
       - Returns all announcements sorted by createdAt desc
       - No parameters

    2. **getActive** (query):
       - Returns announcements where status === 'active' AND:
         - startDate <= current date
         - (endDate is null OR endDate >= current date)
         - targetAudience === 'all' OR targetAudience === 'customers'
       - Use the by_status_startDate index
       - Returns only active, currently-valid announcements

    3. **create** (mutation):
       - Parameters: title, message, type, status, startDate, endDate?, isDismissible, targetAudience
       - Auto-generates createdAt and updatedAt as ISO strings
       - Returns the new announcement ID

    4. **update** (mutation):
       - Parameters: id, plus all editable fields
       - Updates updatedAt to current timestamp
       - Returns void

    5. **remove** (mutation):
       - Parameters: id
       - Deletes the announcement
       - Returns void

    Follow the existing patterns in convex/coupons.ts for mutation structure.
