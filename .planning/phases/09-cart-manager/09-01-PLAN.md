Create `convex/cart.ts` with these functions. All mutations require auth (`getAuthUserId`). Cart data stays embedded in the users table (same schema approach — `v.any()` array on user doc).

**Mutations:**

1. `addItem` — Args: `{ productId: v.id("products"), quantity: v.number() }`. Fetches the product from DB to get current price, stock, name, image. Validates: product exists, product is active (`status !== "discontinued"`), `quantityInStock >= requestedQuantity`. If item already in cart, increments quantity (capped at `quantityInStock`). Builds the CartItem from fresh product data (NOT from client — this prevents price manipulation). Patches user.cart. Returns `{ success: true, item: CartItem }` or `{ success: false, error: string }` (Hebrew error messages like "המוצר אזל מהמלאי").

2. `removeItem` — Args: `{ productId: v.id("products") }`. Filters item from cart array. Patches user.cart.

3. `updateQuantity` — Args: `{ productId: v.id("products"), quantity: v.number() }`. Fetches product to validate stock. If `quantity <= 0`, removes item. If `quantity > quantityInStock`, caps at stock level and returns warning. Patches user.cart.

4. `clearCart` — No args. Sets user.cart to `{ items: [], updatedAt: now }`.

5. `mergeGuestCart` — Args: `{ guestItems: v.array(v.any()) }`. This is the login merge function. For each guest item:
   - Fetch product from DB to validate it still exists, is active, and check current stock/price
   - If product out of stock → add to `skippedItems` array with reason "אזל מהמלאי"
   - If price changed → use new price, add to `priceChanges` array with `{ productName, oldPrice, newPrice }`
   - If quantity exceeds stock → cap at available stock, add to `adjustedItems` array
   - Merge with existing cloud cart (take higher quantity per product, capped at stock)
   - Save merged cart to user doc
   - Return `{ mergedCart: CartItem[], skippedItems, priceChanges, adjustedItems }` — this is `CartSyncResult`

6. `getCart` — Query (not mutation). Returns user.cart items array or empty array if no cart.

**Create `apps/storefront/hooks/useCart.ts`:**

This is the unified cart interface. It detects auth status and routes operations to the correct backend.

```typescript
// Pseudocode structure — implement fully
export function useCart() {
  const user = useQuery(api.users.get);
  const isAuthenticated = user !== null && user !== undefined;
  const isLoading = user === undefined; // auth still loading

  // For authenticated users: Convex is the source of truth
  const convexCart = useQuery(api.cart.getCart, isAuthenticated ? {} : "skip");
  const addItemMutation = useMutation(api.cart.addItem);
  const removeItemMutation = useMutation(api.cart.removeItem);
  const updateQuantityMutation = useMutation(api.cart.updateQuantity);
  const clearCartMutation = useMutation(api.cart.clearCart);
  const mergeGuestCartMutation = useMutation(api.cart.mergeGuestCart);

  // For guests: localStorage + React state
  const [guestCart, setGuestCart] = useState<CartItem[]>(() => loadFromLocalStorage());
  const [guestCoupon, setGuestCoupon] = useState<AppliedCoupon | null>(() => loadCouponFromLocalStorage());

  // Derive current items based on auth status
  const items: CartItem[] = isAuthenticated ? (convexCart?.items ?? []) : guestCart;
  const appliedCoupon = isAuthenticated ? (convexCart?.appliedCoupon ?? null) : guestCoupon;
```

**Cart actions (dispatch-style pattern):**

Each action function checks `isAuthenticated` and routes to the right backend:

- `addItem(product: Product, quantity?: number)` — For authenticated: calls `addItemMutation({ productId: product._id, quantity })`. For guest: builds CartItem from product data (using same logic as current addToCart), validates against `product.quantityInStock`, adds to guestCart state, saves to localStorage. Returns success/error status.

- `removeItem(productId: string)` — Auth: `removeItemMutation({ productId })`. Guest: filter from state + save to localStorage.

- `updateQuantity(productId: string, quantity: number)` — Auth: `updateQuantityMutation({ productId, quantity })`. Guest: update in state + localStorage. If quantity <= 0, remove.

- `clearCart()` — Auth: `clearCartMutation()`. Guest: clear state + localStorage.

- `mergeOnLogin()` — Called internally when auth state transitions from guest to authenticated. Reads localStorage cart, calls `mergeGuestCartMutation({ guestItems })`, clears localStorage, returns `CartSyncResult`. This function should be called from a `useEffect` that watches auth transitions.

**Auth transition effect (inside useCart):**
```typescript
const prevAuthRef = useRef<boolean | null>(null);
useEffect(() => {
  const wasGuest = prevAuthRef.current === false;
  prevAuthRef.current = isAuthenticated;
  
  if (isAuthenticated && wasGuest && guestCart.length > 0) {
    // Guest just logged in with items — merge
    mergeGuestCartMutation({ guestItems: guestCart })
      .then((result) => {
        // Clear localStorage
        localStorage.removeItem('cart');
        setGuestCart([]);
        // Surface sync issues via syncResult state
        if (result.skippedItems.length > 0 || result.priceChanges.length > 0) {
          setSyncResult(result);
        }
      })
      .catch(console.error);
  }
}, [isAuthenticated]);
```

**localStorage adapter functions (private to useCart.ts):**
```typescript
const CART_STORAGE_KEY = 'cart';

function loadFromLocalStorage(): CartItem[] {
  try {
    const saved = localStorage.getItem(CART_STORAGE_KEY);
    if (saved) return JSON.parse(saved).items || [];
  } catch { /* silent */ }
  return [];
}

function saveToLocalStorage(items: CartItem[], coupon: AppliedCoupon | null) {
  try {
    localStorage.setItem(CART_STORAGE_KEY, JSON.stringify({ items, appliedCoupon: coupon }));
  } catch { /* silent */ }
}
```

Simple React context for cart modal state. This replaces the Redux `uiSlice` cart-modal parts.

```typescript
import React, { createContext, useContext, useState, useCallback } from 'react';

interface CartUIContextType {
  isCartOpen: boolean;
  openCart: () => void;
  closeCart: () => void;
  toggleCart: () => void;
}

const CartUIContext = createContext<CartUIContextType | null>(null);

export function CartUIProvider({ children }: { children: React.ReactNode }) {
  const [isCartOpen, setIsCartOpen] = useState(false);
  
  const openCart = useCallback(() => setIsCartOpen(true), []);
  const closeCart = useCallback(() => setIsCartOpen(false), []);
  const toggleCart = useCallback(() => setIsCartOpen(prev => !prev), []);
  
  return (
    <CartUIContext.Provider value={{ isCartOpen, openCart, closeCart, toggleCart }}>
      {children}
    </CartUIContext.Provider>
  );
}

export function useCartUI() {
  const ctx = useContext(CartUIContext);
  if (!ctx) throw new Error('useCartUI must be used within CartUIProvider');
  return ctx;
}
```
